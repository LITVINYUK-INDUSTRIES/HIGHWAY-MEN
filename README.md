# HIGHWAY-MEN
A modular browser-based game engine built with Three.js.

## Current Version
v0.0.5.4-ALPHA

## Description
HIGHWAY-MEN is an experimental open-world driving engine designed for modular development, multi-AI collaboration, and rapid iteration. The project is structured into clean folders for UI, engine, systems, and assets.

## How to Run
1. Open index.html in a local web server (VS Code Live Server recommended).
2. Or enable GitHub Pages in the repository settings.
3. The game loads automatically in the browser.

## Folder Structure
/config  
/engine  
/systems  
/ui  
/assets
